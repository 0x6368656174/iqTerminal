Класс описывающий информацию о пользователе

Данный класс содержит модели и данные, используемые для отображения странички информации о пользователе.

QString UserInfo::userProfileFile
readwrite

Имя файла пользователя

Информация о пользователе грузится из XML-файла, указаного в данном параметре. При изменении имени файла о пользователе, информация автоматически перезагрузится.

Доступ:
userProfileFile(), setUserProfileFile(), userProfileFileChanged()
