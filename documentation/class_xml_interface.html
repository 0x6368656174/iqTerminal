Подробное описание

Класс, описывающий методы расширяющие классы для работы с XML-фалами

Данный класс описывает методы, расщиряющие функционал классов для работы с XML-файлами. Это методы для работы с XML-элементами (findElement(), createElement()), а так же методы, позволяющие AbstractXmlItemsModel загружать свои данные из XML-файлов (reloadModel()) и сохранять свои данные в XML-файл (saveModel()).

Объявления и описания членов классов находятся в файлах:

  • terminalQmlPlugin/xmlinterface.cpp
